[ INSTALLATION ]

FROM CHROME WEB STORE

  1. Visit the Chrome Web Store (link coming soon)
  2. Click "Add to Chrome"
  3. Confirm the installation
  4. Open any YouTube video and click the extension icon

MANUAL INSTALLATION (DEV MODE)

  1. Clone the repository: git clone https://github.com/yourusername/YouTubeTranscriptCopier.git
  2. Run: npm install && npm run build:all
  3. Open Chrome and navigate to chrome://extensions
  4. Enable "Developer mode" (top right)
  5. Click "Load unpacked" and select the project directory
