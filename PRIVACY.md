# Privacy Policy

This extension collects:
- YouTube transcript content for copying purposes
- User-defined prompt templates (synced via Chrome Storage)
- Language preference (English / Japanese, stored locally)
- Theme preference (Clean Modern / Cyberpunk, stored locally)
- Free edit mode text (stored locally)

We do not:
- Collect personal information
- Share any data with third parties
- Use data for purposes other than the extension's functionality

All data is stored in your browser only and is never sent to external servers.

Contact: [Issues](https://github.com/infoHiroki/YouTubeTranscriptCopier/issues)
